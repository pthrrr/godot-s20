# godot-s20 (Split Screen)

3D arena shooter with split screen two-player mode. Based on [gdquest first_3d_game_godot4_arena_fps](https://www.gdquest.com/library/first_3d_game_godot4_arena_fps/).

## Features

- Split screen local multiplayer (2 players)
- Keyboard + gamepad controls
- Bat mobs that chase the nearest player
- 2 mob spawners, max 20 living mobs
- 30-second round timer
- Score tracking per player, winner announced at round end
- Kill plane with player respawn and score reset

## Branches

- `main` — single player mode
- `two_player_split_screen` — split screen multiplayer (this branch)

## Requirements

- Godot 4.6
